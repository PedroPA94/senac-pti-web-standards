# Tinta Certa Pinturas

Landing page desenvolvida como parte da disciplina **Web Standards** do curso de **Análise e Desenvolvimento de Sistemas (SENAC)**.

O projeto consiste em uma página única e responsiva desenvolvida para um profissional autônomo, com o propósito de apresentar seus serviços e estimular o contato de potenciais clientes.

O trabalho teve como **objetivo** a criação de uma **landing page completa utilizando HTML, CSS e Flexbox**, aplicando boas práticas de estrutura, semântica e layout.
